# Road Accident Dashboard Data Analysis (Interactive Dashboard creation using MS Excel)
## Project Objective
To develop an interactive Excel dashboard that transforms road accident data into meaningful insights through data visualization, enabling stakeholders to monitor casualty trends, identify risk factors, and support informed road safety decisions.

## Dataset Used
 - <a href="https://github.com/Oluwakorey/Data-Analysis-Dashboard/blob/main/dft-road-casualty-statistics-casualty-provisional-mid-year-unvalidated-2022%20(1).csv"> Road accident Data </a>

## Question KPIs
- What is the total number of road accident casualties?
- How many casualties were Fatal, Serious, and Slight?
- Which vehicle type recorded the highest number of casualties?
- How do casualty figures compare month-by-month between 2021 and 2022?
- Which road type has the highest number of casualties?
- How do road surface conditions affect casualty numbers?
- Are more casualties recorded during daylight or night?
- Do urban areas record more casualties than rural areas?
- How do casualty trends change over time?
- How can interactive filters (Year, Accident Date, and Location) help users explore accident patterns?

## Project Process
# 1. Data Preparation
Imported the road accident dataset into Microsoft Excel.
Reviewed the dataset for completeness and consistency.
Cleaned missing values and standardized data fields where necessary.
Converted the dataset into an Excel Table for easier management and analysis.
# 2. Data Modeling
Created multiple Pivot Tables to summarize casualty information.
Organized accident data by:
Severity
Vehicle type
Road type
Road surface
Light condition
Urban/Rural location
Month and Year
# 3. Dashboard Development
Built Pivot Charts to visualize accident patterns.
Designed KPI cards to display key casualty figures.
Added slicers and timelines for interactive filtering by:
Year
Accident Date
Location
Applied a consistent dashboard layout with color coding for better readability and user experience.
# 4. Dashboard Design
Used a dark-themed dashboard for improved visual appeal.
Arranged visualizations logically for quick interpretation.
Highlighted critical metrics using large KPI cards and percentage indicators.
